### Основы юнит-тестирования в JS

## Порядок установки
Клонировать репозиторий к себе:
```
    git clone https://github.com/aboronilov/react_unit_test
```
перейти в папку проекта, установить зависимости:
```
    cd test-app
    npm i
```
запустить проект:
```
    npm start
```
запустить тесты:
```
    npm test
```
